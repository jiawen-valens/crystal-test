# 水晶人格测试 - 命定你的能量水晶

## 介绍
10道题，测试你的专属人格与能量水晶。页面精致适配手机，直接引导微信购买，主打情绪疗愈和命定感。

## 本地运行
1. 下载（或 clone）本项目到本地
2. 进入目录，双击 `index.html` 或用任意静态服务器启动即可
   - VSCode 推荐 Live Server 插件 (`右键 index.html -> Open with Live Server`)
   - 或命令行 `npx serve .`（需[Node.js](https://nodejs.org)环境）

## 部署到 Zeabur
1. 登录 [Zeabur](https://zeabur.com)
2. 新建项目，上传代码或连接 GitHub Repo
3. 选择「静态网站」类型，入口目录留空（主目录）
4. 点发布即可，等待几秒自动生成你的专属域名！
